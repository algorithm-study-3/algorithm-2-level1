# 📝 알고리즘 문제 풀이 - 2025-03-14

## 문제 📖

-   **문제 링크:** [리트코드 - Palindrome Number](https://leetcode.com/problems/palindrome-number/)

-   **문제 유형:** `Math`

-   **난이도:** EASY

-   **풀이 시간:** ⏳ 1분 - 실제 걸렸던 시간

## ✍ 접근 방식

1. 문자열을 가지고 와서 배열로 만든 뒤 뒤집는다.
2. 그 후 다시 문자열로 만들어준다.
3. 문자열과 새로 생성한 palindrome이 같은지 확인 후 같으면 true 아니면 false를 반환한다.
